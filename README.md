# NeoGeoCD-metal-slug-sprite-tool
Convert *.SPR file from Neo Geo CD version Metal Slug / Metal Slug 2 to TiledGGD readable tile.

Code is generated by AI.

Also thanks to [NeoGeo Development Wiki](https://wiki.neogeodev.org/index.php?title=Sprite_graphics_format)

# Usage
```
neocd_spr.exe <input.SPR> <output>
